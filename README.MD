How to run the app;

1. Clone the repo:
   git clone https://github.com/rihan-a/s-task.git

2. Start the backend:
   cd backend
   npm install
   npm run dev

    This will start the backend at:
    http://localhost:8000/api/report

3. Start the frontend:
   cd frontend
   npm install
   npm run dev

This will start the frontend at:
http://localhost:5173
